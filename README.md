**🚀 CurricuForge**
**Generative AI–Powered Curriculum Design System**


CurricuForge is an AI-powered curriculum design platform that automatically generates structured academic programs using Generative AI (Gemini). The system allows educators and institutions to design industry-aligned curricula by simply providing a few educational parameters.

Instead of manually planning entire academic programs, CurricuForge intelligently generates courses, topics, and learning outcomes within seconds.

**📌 Problem Statement**

Designing academic curricula that align with modern industry needs is a complex and time-consuming process. Educational institutions must frequently update their programs to keep up with evolving technologies and skill demands.

CurricuForge addresses this challenge by automating curriculum design using Generative AI, enabling faster, smarter, and more scalable educational planning.

**💡 Solution Overview**

CurricuForge generates complete curriculum structures based on user inputs such as:

Skill or domain

Education level

Number of semesters

Weekly learning hours

Industry focus

Using these inputs, the system produces:

Semester-wise course structures

Course descriptions

Topic recommendations

Learning outcomes

The generated curriculum follows a logical progression from fundamentals to advanced topics.

4️⃣ Output Module

**⚡ Installation**

Clone the repository

**git clone https://github.com/JohnSamuel85/CurricuForge-GenaiForge.git
cd curricuforge**

Create virtual environment

**python -m venv venv**

Activate environment

Windows

**venv\Scripts\activate**

Mac/Linux

**source venv/bin/activate**

Install dependencies

**pip install -r requirements.txt**

▶️ Running the Project

Start the backend server (via VS CODE Terminal)

**uvicorn backend.app.main:app --reload**

Open in browser
